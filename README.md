# Telegram Bot & VK Data Parser API 🚀

Учебный проект асинхронного веб-сервиса и чат-бота для сбора аналитики из социальных сетей.

## 🛠 Стек технологий
* **Python 3.13**
* **FastAPI** (Асинхронный веб-фреймворк для API)
* **vk_api** (Интеграция с ВКонтакте API)
* **python-dotenv** (Безопасное управление конфигурацией и токенами)
* **Git / GitHub / МосХаб**

## ⚡ Функционал
1. **Главная страница:** Проверка работоспособности сервера.
2. **Эндпоинт `/parse/{group_domain}`:** Принимает короткое имя любого открытого сообщества VK, обращается к API ВКонтакте, фильтрует данные и возвращает чистый JSON с метриками (ID поста, лайки, просмотры, превью текста) для последних 3 постов.

## 🔒 Безопасность
Все конфиденциальные данные (VK_TOKEN) вынесены в переменные окружения (`.env`) и защищены от попадания в публичный репозиторий с помощью `.gitignore`.

## 👨‍💻 Особенности разработки
Код написан вручную (без слепого копирования ИИ-генераций) для глубокого освоения синтаксиса FastAPI и понимания жизненного цикла HTTP-запросов.
